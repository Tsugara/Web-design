Web Design Basics

The main steps:

  • Define Internet audience
    Colors, images, fonts, and layout should be guided by the preferences of your audience. Site Content Type (reading level, the number of animations, etc.) must correspond to the selected audience.
  • Determine the purpose of the site
    Some common goals Websites: be informative, create a personal home page, set the corporate web presentation, e-commerce business.
  • Determine the total content and create a site map
    Sitemap sometimes called a storyboard. Standard structure of the organization Web sites: hierarchical, linear and random.
  • Sketch out a preliminary layout
    Stick to consistency in their layouts & mdash; color, font, location, logo, navigation bar, etc. Design the layout of the home page so as to avoid scrolling. Make a layout of internal pages consistent, with easy navigation.
